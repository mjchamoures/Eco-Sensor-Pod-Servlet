Detailed Description

BACKEND: SensorDataRetriever Class

Purpose: Contains all the DataTurbine retrieval tasks. Listens for requests from the DataFacilitator Servlet. Fetches data for the requested channel from the requested sensor pod deployment

Author: Michael Chamoures

Date: 6/11/2013
