Overall Sensor Pod System.

The OSDT Sensor Pod is a waterproof buoy system that interfaces with external sensors in the water and communicates with shore-side computers where the data is displayed via charts. This provides real-time situational awareness as well as control.

Project Abstract

Redesigned Harware System

The physical components of the system have been redesigned to include a clean and compact PCB sensor system connected via PWB traces to collect various environmental measurements

New User-Friendly Web Application

A newly created web application that provides the ability to view both past and live, real-time data from various deployed sensor pods around the world
